## API Reference

### Defaults

Your can override the component default props in `provider.tsx`.

::code-group

```ts [provider.tsx]
export const defaultConfig: Config = {
  BaseAccordion: {
    rounded: 'sm',
    action: 'dot',
    color: 'default',
    dotColor: 'primary',
  },
}
```
