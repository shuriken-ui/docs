## API Reference

### Defaults

Your can override the component default props in `app.config.ts`.

::code-group

```ts [app.config.ts]
export default defineAppConfig({
  nui: {
    BaseTabSlider: {
      justify: 'start',
      size: 'md',
      rounded: 'lg',
      color: 'primary',
    },
  }
})
```
