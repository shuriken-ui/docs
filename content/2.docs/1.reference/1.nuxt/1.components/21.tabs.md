## API Reference

This component has props that you can use to modify its visual style.

:component-meta{name="BaseTabs"}

### Defaults

Your can override the component default props in `app.config.ts`.

::code-group

```ts [app.config.ts]
export default defineAppConfig({
  nui: {
    BaseTabs: {
      type: 'tabs',
      justify: 'start',
      color: 'primary',
    },
  }
})
```
::
