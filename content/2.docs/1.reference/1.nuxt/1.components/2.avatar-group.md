## API Reference

This component has props that you can use to modify its behavior.

:component-meta{name="BaseAvatarGroup"}

### Defaults

Your can override the component default props in `app.config.ts`.

::code-group

```ts [app.config.ts]
export default defineAppConfig({
  nui: {
    BaseAvatarGroup: {
      limit: 4,
      size: 'sm',
    },
  }
})
```
