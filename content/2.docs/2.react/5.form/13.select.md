### Defaults

Your can override the component default props in `provider.config.ts`.

::code-group

```js [provider.config.ts]
export default providerConfig({
  nui: {
    BaseSelect: {
      rounded: 'sm',
      size: 'md',
      contrast: 'default',
    },
  }
})
```
