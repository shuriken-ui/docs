## API Reference

This component has props that you can use to modify its behavior.

### Defaults

Your can override the component default props in `provider.config.ts`.

::code-group

```ts [provider.config.ts]
export default providerConfig({
  nui: {
    BaseAccordion: {
      rounded: 'sm',
      action: 'dot',
    },
  }
})
```
