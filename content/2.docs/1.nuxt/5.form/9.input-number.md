## API Reference

This component has props that you can use to modify its visual style and behavior.

:component-meta{name="BaseInputNumber"}

### Defaults

Your can override the component default props in `app.config.ts`.

::code-group

```js [app.config.ts]
export default defineAppConfig({
  nui: {
    BaseInputNumber: {
      inputmode: 'numeric',
      rounded: 'sm',
      size: 'md',
      contrast: 'default',
    },
  }
})
```
::

### Steps

Use the `step` prop to handle the incremental steps of the input.

::code-group{expandable}

```js [DemoInputNumberSteps.vue]
<template>
  <BaseInputNumber
    v-model="fields.first"
    rounded="md"
    label="Temperature (°C)"
    icon="carbon:temperature-celsius"
    :min="-20"
    :max="20"
    :step="0.05"
  />

  <BaseInputNumber
    v-model="fields.second"
    rounded="md"
    label="Temperature (°F)"
    icon="carbon:temperature-fahrenheit"
    :min="0"
    :step="0.5"
  />

  <BaseInputNumber
    v-model="fields.third"
    rounded="md"
    label="BTC"
    icon="ic:sharp-currency-bitcoin"
    :min="0"
    :step="0.000001"
  />

  <BaseInputNumber
    v-model="fields.fourth"
    rounded="md"
    icon="material-symbols-light:encrypted-outline"
    label="Bytes"
    :min="0"
    :step="1024"
  />
</template>

const fields = reactive({
  first: 0.15,
  second: 1.5,
  third: 1.542312,
  fourth: 1024,
})
```
