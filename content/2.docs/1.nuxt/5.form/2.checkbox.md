### True/False values

Use the `true-value` and `false-value` props to toggle the state of the checkbox.

::code-group

```js [DemoCheckboxTrueFalse.vue]
<template>
  <BaseCheckbox
    v-model="value"
    name="true-false"
    :label="value"
    :true-value="'checked'"
    :false-value="'not-checked'"
  />
</template>

<script setup lang="ts">
const value = ref('not-checked')
</script>
```
